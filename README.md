#  Dynamic Quote Generator

This is a dynamic web application built with **vanilla JavaScript**, focusing on:

-  Advanced DOM Manipulation  
-  Web Storage (localStorage & sessionStorage)  
-  JSON Import/Export  
-  Dynamic Content Filtering  
-  Simulated Server Sync & Conflict Handling  

Created as part of the **ALX Front-End JavaScript Second Chance Project**, this app allows users to manage, view, filter, and sync inspirational quotes.

---

##  Features

-  **Random Quote Generator** ‚Äì Shows a new quote from a selected or all categories.
-  **Add New Quotes** ‚Äì Users can dynamically add quotes and categories via form input.
-  **Persistent Storage** ‚Äì All quotes are stored in localStorage to survive page reloads.
-  **Import/Export** ‚Äì Export your quotes to JSON or import from a `.json` file.
-  **Category Filter** ‚Äì Filter quotes by category and remember the last filter selected.
-  **Server Sync (Mock)** ‚Äì Simulates syncing with a remote server and merging data.
